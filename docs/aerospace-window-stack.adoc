== Synopsis
[verse]
// tag::synopsis[]
aerospace window-stack [-h|--help] [--window-id <window-id>]

// end::synopsis[]

== Description

// tag::body[]
{manpurpose}

This command currently works only when the focused window already occupies its own top-level column in `niri` layout.
If the left neighbour is already a vertical stacked column, the window is appended to that stack.
Otherwise, the left neighbour is wrapped into a vertical column and the focused window is added to it.
